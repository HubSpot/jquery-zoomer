What is jQuery Zoomer?

jQuery Zoomer is a library which lets you zoom up your iframes using CSS transforms.

Browser support

Tested and working: Chrome, FF3.6+, IE8+

Usage

$('iframe').zoomer({ width: 200, zoom: 0.5 });
